RecipesApp
=============

Aplikacja webowa napisana w Angular 4 do tworzenia bazy przepisów kuchennych w raz z tworzeniem nowych proporcji składników.

### Instalacja i konfiguracja
1. cd recipes-app
2. ng serve --open

### Wykorzystane technologie ###

* Angular 4
* CSS 3
* Bootstrap
* HTML 5
